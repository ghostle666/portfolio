Aquarium Landing page (case study)

Project Overview

Design an engaging, infor mative, and visually immersive website for a fictional aquarium center, aimed at attracting more visitors, showcasing upcoming events, and providing essential visitor information.

Tools Used: Figma.

Objective

Create a modern, accessible, and visually engaging website that communicates the wonder of marine life and encourages users to visit or book tickets. The primary goals were to:

  • Increase ticket conversions
  • Communicate mission-driven values (education, conservation, exploration)
  • Improve access to event and exhibit information

User flow

  1. Homepage - Stunning hero image, brand message, and ticket CTA
  2. About Page - Outlines mission and educational impact
  3. Events Page - Upcoming events with descriptions and schedules
  4. Exhibits Page - High-impact visuals and informative exhibit sections
  5. Footer - Quick links, contact, and social media
